1. Database Creation

First, create three separate tables: temperatures, countries, and stations table. To do so, I wrote a function named prepare_df to prepare the temperature dataset such that the data is easier to understand.

  • The temperatures table will contain station ID, year of measurement, month of measurement, and average temperature.
  • The countries table will contain country names and their corresponding country codes.
  • The stations table will contain station ID, latitude, longitude, elevation, and the name of the station.
def prepare_df(df):
    df = df.set_index(keys=["ID", "Year"])
    df = df.stack()
    df = df.reset_index()
    df = df.rename(columns = {"level_2"  : "Month" , 0 : "Temp"})
    df["Month"] = df["Month"].str[5:].astype(int)
    df["Temp"]  = df["Temp"] / 100
    df["NewID"] = df["ID"].str[0:2]
    return(df)
with sqlite3.connect("HW1.db") as conn: # create a database in current directory called temps.db
    df_iter = pd.read_csv("temps.csv", chunksize = 100000)
    for i, df in enumerate(df_iter):

    url = "station-metadata.csv"
    stations = pd.read_csv(url)
    stations["NewID"] = stations["ID"].str[0:2]
    stations.to_sql("stations", conn, if_exists = "replace", index=False)
    countries = pd.read_csv("countries.csv")
    countries = countries.rename(columns={"FIPS 10-4": "NewID"})
    countries.to_sql("countries", conn, if_exists = "replace", index=False)

2. Query Function

Below I created a query function query_climate_database which returns a dataframe based off of filters. The parameters specify the country, the year we start from, the year we end at, and the month we wish to analyze. The dataframe displays station name, latitude, longitude, and average temperature of the station during the month and year.

def query_climate_database(db_file, country, year_begin, year_end, month):
    """
    Query function to filter climate data 
    
    Args:
    country: country of interest 
    year_begin: start year desired 
    year_end: end year desired 
    month: a specific month 
    
    Return:
    Dataframe of filtered data 
    """
    with sqlite3.connect(db_file) as conn:
        cmd = \
        f"""
        SELECT S.Name, S.LATITUDE, S.LONGITUDE, C.Name "Country", T.Year, T.Month, T.Temp
        FROM stations S 
        LEFT JOIN countries C ON C.NewID = S.NewID
        RIGHT JOIN temperatures T ON S.ID = T.ID 
        WHERE T.Year <= {year_end} AND T.Year >= {year_begin} AND T.Month = {month} AND C.Name = "{country}"
        """ 
        df = pd.read_sql_query(cmd, conn)
        return(df)

3. Geographic Mapbox Scatterplot

We use query_climate_database to create another function temperature_coefficient plot which answers how the average yearly change in temperature varies within a country. The function accepts the same parameters, as well as min_obs which will filters stations which do not have the specified number of data points. We also add **kwargs to customize our mapbox scatterplot we made using px.scatter_mapbox().

Inside our function, I wrote another function coef which conducts linear regression in order to compute how each station changes in average temperature over time.

def temperature_coefficient_plot(db_file, country, year_begin, year_end, month, min_obs, **kwargs):
    """
    Construct a geographic mapbox scatterplot of yearly temperature increase
    
    Args:
    country: desired country to display 
    year_begin: desired start year 
    year_end: desired end year 
    month: a specific month 
    min_obs: minimum required observations of station
    **kwargs: additional keyword arguments
    
    Return:
    Scatter_mapbox plot
    """
    import calendar 
    def coef(data_group):
        x = data_group[["Year"]] # 2 brackets because X should be a df
        y = data_group["Temp"]   # 1 bracket because y should be a series
        LR = LinearRegression()
        LR.fit(x, y)
        return LR.coef_[0]
    df = query_climate_database(db_file, country, year_begin, year_end, month)
    test = df.groupby(["NAME"]).size()
    names = test[test >= min_obs].index
    df = df[df.NAME.isin(names)]
    coefs = df.groupby(["NAME", "Month"]).apply(coef).round(4)
    plot_df = pd.DataFrame(coefs)
    merged_df = pd.merge(df, plot_df, on='NAME')
    merged_df = merged_df.rename(columns={0: "Estimated Yearly Increase (Celcius)"})
    fig = px.scatter_mapbox(merged_df, 
                            lat = "LATITUDE",
                            lon = "LONGITUDE", 
                            hover_name="NAME", 
                            color_continuous_midpoint = 0,
                            color = "Estimated Yearly Increase (Celcius)",
                            title = (f"""Estimates of yearly increase in {calendar.month_name[month]} 
                             <br>temperature for stations in {country}, from {year_begin} to {year_end}"""),
                            **kwargs)
    return(fig)

                                   color_continuous_scale = color_map)

4. Additional Query Function and Plotly Boxplot and Lineplot

I decided to write a query function which filters out only the equator countries. The reason why I did this was because I believe that it is valuable to see how the temperature changes each year from these equator countries because it may illustrate that even at the hottest parts of our world, the temperature increases, potentially infering global warming.

def query_equator(db_file, year_begin, year_end):
    """
    Query function of only the Equator countries
    
    Args:
    country: desired country to display 
    year_begin: desired start year 
    year_end: desired end year
    
    Return:
    Dataframe with desired filters 
    """
    with sqlite3.connect(db_file) as conn:
        cmd = \
        f"""
        SELECT S.Name, S.LATITUDE, S.LONGITUDE, C.Name "Country", T.Year, T.Month, T.Temp
        FROM stations S 
        LEFT JOIN countries C ON C.NewID = S.NewID
        RIGHT JOIN temperatures T ON S.ID = T.ID 
        WHERE T.Year <= {year_end} AND T.Year >= {year_begin} AND S.LATITUDE >= -5 AND S.LATITUDE <= 5
        AND (C.Name = "Congo, Democratic Republic of the" OR C.Name = "Gabon" 
        OR C.Name = "Somalia" 
        OR C.Name = "Congo, Republic of the" OR C.Name = "Uganda" 
        OR C.Name = "Maldives" OR C.Name = "Indonesia" OR C.Name = "Kiribati" 
        OR C.Name = "Sao Tome and Principe" OR C.Name = "Kenya" 
        OR C.Name = "Ecuador" OR C.Name = "Colombia" OR C.Name = "Brazil")
        """ 
        df = pd.read_sql_query(cmd, conn)
        return(df)

The temperature_box_plot function below addresses how the average temperature of countries on the equator change year to year. I made sure that the function uses the same station for every year such that the data is not unevenly weighted. The plot which this function produces is important because we can see if our temperature is rising or lowering as time goes on. And as we can see from the figure below, there is a slight gradual increase in the boxplot medians of Brazil, providing visual evidence that temperatures are increasing for equator countries.

def temperature_box_plot(db_file, country, year_begin, year_end, **kwargs):
    """
    Construct boxplot of yearly average temperatures of specified equator country 
    
    Args:
    country: desired country to display 
    year_begin: desired start year 
    year_end: desired end year 
    
    Return:
    px.box plot of yearly average temperatures of specified equator country
    """
    df = query_equator(db_file, year_begin, year_end)
    df = df[df.Country == country]
    new = df.groupby(["NAME", "Year"])["Temp"].aggregate(np.mean)
    test = df.groupby(["NAME", "Year"]).size()
    test = test.reset_index()
    names = test.groupby(["NAME"]).size() >= year_end - year_begin + 1 
    names = names[names].index
    plot_df = pd.DataFrame(new)
    plot_df = plot_df.reset_index()
    plot_df = plot_df[plot_df.NAME.isin(names)]
    fig = px.box(df,
                 x = "Country",
                 y = "Temp",
                 color = "Year",
                 title = f"Boxplots of {country}'s Average Temperature by Year from {year_begin} to {year_end}",
                 category_orders={"Year": [ix for ix in range(year_begin, year_end)]},
                 **kwargs
                )
    return fig
fig = temperature_box_plot(db_file, "Brazil", 1980, 2020)
fig.show()

I constructed the temperature_lineplot_equator function to display a lineplot of how the temperature changes over time in each month, and each colored line on the lineplot represents a different station. I felt this was important because like the boxplot it was another way to visualize how there is an increase in temperature in equator countries. For instance, as we can see below the Average Temperature of Station in Indonesia from 1985 to 2020 increased for all 4 stations plotted.

def temperature_lineplot_equator(db_file, year_begin, year_end, country, month, **kwargs):
    """
    Construct lineplot of yearly average temperatures of specified equator country in month
    
    Args:
    country: desired country to display 
    year_begin: desired start year 
    year_end: desired end year 
    month: specific month of year
    **kwargs: keyword arguments for lineplot
    
    Return:
    px.line plot of yearly average temperatures of specified equator country
    """
    import calendar 
    df = query_equator(db_file, year_begin, year_end)
    df = df[df.Month == month]
    df["Date"] = pd.to_datetime(df["Year"].astype(str) + "-" + df["Month"].astype(str))
    df = df[df.Country == country]
    new = df.groupby(["NAME", "Date"])["Temp"].aggregate(np.mean)
    test = df.groupby(["NAME", "Date"]).size()
    test = test.reset_index()
    names = test.groupby(["NAME"]).size() >= (year_end - year_begin) - 5  
    names = names[names].index
    plot_df = pd.DataFrame(new)
    plot_df = plot_df.reset_index()
    plot_df = plot_df[plot_df.NAME.isin(names)]
    fig = px.line(data_frame = plot_df,
                  x = "Date",
                  y = "Temp",
                  color = "NAME",
                  title = (f"""Average Temperature of Station in {country} <br>
                           by Year in {calendar.month_name[month]} from {year_begin} to {year_end}"""),
                  )
    return(fig)
plot = temperature_lineplot_equator(db_file, 1985, 2020, "Indonesia", 6)
plot.show()

5. Concluding Remarks

As we can see from the query functions and graphics constructed, there is overwhelming evidence for global warming. The mapbox plot, boxplot, and line plots all show that there is a majority increase in temperature, and this conclusion was able to be realized because the visualizations helped to easily explain the data.
